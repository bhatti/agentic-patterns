# Pattern 46: Goal Setting and Monitoring (Agentic)

## Overview

**Goal monitoring** is the **executive** **layer**: **SMART** **objectives**, **metrics**, **deviation** **detection**, and **strategy** **updates** when reality diverges. It complements **prioritization** (43)—which **orders** **tasks**—by asking **whether** **we’re** **achieving** **the** **right** **outcomes**.

**Gulli**: [goal-monitor.md](https://github.com/anti-achismo-social-club/subagents-design-patterns/blob/main/agents/goal-monitor.md).
