# Pattern 50: Exploration and Discovery (Agentic)

## Overview

**Exploration** balances **discovering** **new** **information** **or** **actions** **vs** **exploiting** **known** **good** **policies**—in **agents**, **research**, **tool** **use**, **and** **open-ended** **search**. Methods range from **systematic** **sweeps** **to** **curiosity**-**weighted** **next** **steps** **(often** **paired** **with** **LATS/ToT** **—** **Patterns** **14/41**).
