# Pattern 48: Inter-Agent Communication (Agentic / A2A)

## Overview

**Inter-agent communication** provides **structured** **messages**, **routing**, **correlation** **ids**, and **capability** **discovery** so **distributed** **agents** **coordinate** **without** **ad-hoc** **string** **passing**. Aligns with **A2A**-style **protocols** **(Google** **A2A** **spec**, **etc.)**.

**Gulli**: [agent-communicator.md](https://github.com/anti-achismo-social-club/subagents-design-patterns/blob/main/agents/agent-communicator.md).

## Relationship to Pattern 23

**Multi-agent collaboration (23)** covers **roles** **and** **crews**; **Pattern** **48** **emphasizes** **the** **message** **fabric** **(bus**, **schemas**, **async** **handling)**.
